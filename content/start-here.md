# 標題與段落

Markdown 支援六個層級的標題，使用 `#` 符號表示，數量越多層級越低：

## 第二層標題（H2）

使用 `## 標題文字` 建立。適合用於章節標題。

### 第三層標題（H3）

使用 `### 標題文字` 建立。適合用於小節標題。

#### 第四層標題（H4）

使用 `#### 標題文字` 建立。適合用於更細的分類。

##### 第五層標題（H5）

使用 `##### 標題文字` 建立。較少使用，適合深層結構。

###### 第六層標題（H6）

使用 `###### 標題文字` 建立。最小的標題層級。

## 文字格式

這篇用來驗證 Markdown 渲染是否正確，並示範常見語法：

- **粗體**：使用 `**文字**` 或 `__文字__`
- *斜體*：使用 `*文字*` 或 `_文字_`
- ***粗斜體***：使用 `***文字***`
- `inline code`：使用反引號 `` `code` ``
- ~~刪除線~~：使用 `~~文字~~`
- [超連結](https://example.com)：使用 `[文字](網址)`

---

## 引用區塊

> 引言：可以包含 **粗體**、`inline code`，也可換行。
> 第二行引言。

使用 `>` 符號建立引用區塊，常用於引述他人文字或強調重點。

## 清單

### 無序清單

使用 `-`、`+` 或 `*` 開頭：

- 無序列表示例
  - 巢狀項目 A
  - 巢狀項目 B
- ✅ 完成項目
- ⏳ 進行中

### 有序清單

使用數字加點號 `1.` 開頭：

1. 有序列表第一步
2. 第二步
   1. 子步驟 2.1
   2. 子步驟 2.2
3. 第三步

## 表格

使用 `|` 和 `-` 建立表格，第二行用於對齊設定：

| 語法 | 說明 | 備註 |
| ---- | ---- | ---- |
| `**bold**` | 粗體 | 強調重點 |
| `*italic*` | 斜體 | 較弱強調 |
| `` `code` `` | 行內程式碼 | 片段 |

對齊方式：
- 靠左：`| ---- |`
- 置中：`| :----: |`
- 靠右：`| ----: |`

## 程式碼區塊（多語言）

### JavaScript

使用三個反引號 ` ``` ` 包裹程式碼，並指定語言以啟用語法高亮：

```js []
function greet(name) {
  const msg = `Hello, ${name}!`;
  console.log(msg);
  return msg;
}

greet("World");
```

### Python

```python []
def fib(n):
    a, b = 0, 1
    out = []
    for _ in range(n):
        out.append(a)
        a, b = b, a + b
    return out

print(fib(7))
```

### Bash

```bash
#!/usr/bin/env bash
echo "Build project"
npm run build
```

### HTML

```html
<section class="card">
  <h2>Card Title</h2>
  <p>Some content here.</p>
</section>
```

### CSS

```css
.card {
  border: 1px dashed #ccc;
  padding: 12px;
}
```

### JSON

```json
{
  "title": "Markdown Test",
  "tags": ["Markdown", "Test"],
  "ok": true
}
```

### Diff（差異顯示）

```diff
- const mode = "dev";
+ const mode = process.env.NODE_ENV || "dev";
```

## 水平分隔線

使用三個或以上的 `-`、`*` 或 `_` 建立分隔線：

```markdown
---
***
___
```

## 內文排版

段落之間要有空行，以確保正確換行。行內 `code` 與外框程式碼區塊需使用適當語言標記，才能看到高亮效果。
